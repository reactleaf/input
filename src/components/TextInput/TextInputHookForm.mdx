# \<TextInput />

TextInput is a `react-hook-form` support version of TextInput. \
This component simply extends `<input type="text" />` with styles

<Canvas of={Stories.HookForm} />

### Usage

```tsx
import { useForm, FormProvider } from "react-hook-form"
import { TextInput } from "@reactleaf/input/hookform"

function FormComponent() {
  const form = useForm()

  return (
    <FormProvider {...form}>
      <TextInput name="name" label="Name" />
    </FormProvider>
  )
}
```

### Extended Props

This components extended from raw component. Just add some props for `<Controller />` such as `name` or `rules`

- `value` - is excluded. `value` is handled by `react-hook-form`.
