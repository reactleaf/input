# \<TextInput />

TextInput is a basic component of `@reactleaf/input` \
This component simply extends `<input type="text" />` with styles

<Canvas of={Stories.Basic} />
<Canvas of={Stories.Disabled} />

### Usage

```tsx
import { TextInput } from "@reactleaf/input"

function FormComponent() {
  const [value, setValue] = useState("")

  return <TextInput label="Name" value={value} onValueChange={(v) => setValue(v)} />
}
```

### Extended Props

`<TextInput />` extends all props of `<input />`
and supports more props below.
