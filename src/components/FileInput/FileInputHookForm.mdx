# \<FileInput />

FileInput is a `react-hook-form` support version of FileInput.

<Canvas of={Stories.HookForm} />

### Usage

```tsx
import { useForm, FormProvider } from "react-hook-form"
import { FileInput } from "@reactleaf/input/hookform"

function FormComponent() {
  const form = useForm()

  return (
    <FormProvider {...form}>
      <FileInput.HookForm name="image" label="Image" maxFileSize={10 * 1000 * 1000} />
    </FormProvider>
  )
}
```

### Extended Props

This components extended from raw component. Just add some props for `<Controller />` such as `name` or `rules`
