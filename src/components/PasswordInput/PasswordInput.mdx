# \<PasswordInput />

PasswordInput is a basic component of `@reactleaf/input` \
This component simply extends `<input type="password" />` with styles

<Canvas of={Stories.Basic} />
<Canvas of={Stories.Disabled} />

### Usage

```tsx
import { PasswordInput } from "@reactleaf/input"

function FormComponent() {
  const [value, setValue] = useState("")

  return <PasswordInput label="Password" value={value} onValueChange={(v) => setValue(v)} />
}
```

### Extended Props

`<PasswordInput />` extends all props of `<input />` excepts `type`.
