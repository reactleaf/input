# \<PasswordInput.HookForm />

PasswordInput.HookForm is a `react-hook-form` support version of PasswordInput. \
This component simply extends `<input type="password" />` with styles

<Canvas of={Stories.HookForm} />

### Usage

```tsx
import { useForm, FormProvider } from "react-hook-form"
import { PasswordInput } from "@reactleaf/input"

function FormComponent() {
  const form = useForm()

  return (
    <FormProvider {...form}>
      <PasswordInput.HookForm name="name" label="Name" />
    </FormProvider>
  )
}
```

### Extended Props

`<PasswordInput.HookForm />` is almost same with `<PasswordInput />`
and also receive props for `<Controller />` such as `name` or `rules`

- `value` - is excluded. `value` is handled by `react-hook-form`.
