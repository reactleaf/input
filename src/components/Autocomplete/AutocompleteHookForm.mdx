# \<Autocomplete />

`@reactleaf/input` provides `react-hook-form` supported version of Autocomplete.

<Canvas of={Stories.HookForm} />

### Usage

```tsx
import { useForm, FormProvider } from "react-hook-form"
import { Autocomplete } from "@reactleaf/input/hookform"

function FormComponent() {
  const form = useForm()

  // options requires label and value field
  const options = ["Option 1", "Option 2", "Option 3"].map((label) => ({ label, value }))

  return (
    <FormProvider {...form}>
      <Autocomplete name="option" label="Option" options={options} />
    </FormProvider>
  )
}
```

### Types

checkout the [react-select documentation](https://react-select.com/props) for more details.

### Extended Props

`<Autocomplete />` extends almost all props of `<Creatable />`. And add some props for `<Controller />` such as `name` or `rules`
