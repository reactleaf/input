# \<Autocomplete />

Autocomplete overrides `react-select`'s `<Creatable>` components with `@reactleaf`'s style.

So, almost every props from `react-select` is available, includes `isMulti`.

### Usage

```tsx
import { Autocomplete } from "@reactleaf/input"

function FormComponent() {
  // options requires label and value field
  const options = ["Option 1", "Option 2", "Option 3"].map((label) => ({ label, value }))
  return <Autocomplete label="Name" options={options} />
}
```

### Types

checkout the [react-select documentation](https://react-select.com/props) for more details.
