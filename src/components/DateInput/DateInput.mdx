# \<DateInput />

DateInput overrides `react-datepicker` components with `@reactleaf`'s style.

So, almost every props from `react-datepicker` is available.

> **Note** : `react-datepicker`'s portal is not available thrhough Storybook's canvas.\
> See the Playground to see how `portal` works.

### Usage

```tsx
import { DateInput } from "@reactleaf/input"

function FormComponent() {
  const [value, setValue] = useState<Date | null>(null)

  return <DateInput label="Name" selected={value} onChange={setValue} />
}
```
