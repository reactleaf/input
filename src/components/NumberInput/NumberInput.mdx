# \<NumberInput />

NumberInput is a special input to handle number input.

> **WARNING** \
> `<NumberInput />` is based on `parseFloat()` \
> Numbers that JavaScript cannot handle are not supported. \
> ex. 123,456,789,012,345,678 -> 123,456,789,012,345,700 \
> ex. 0.1234567890123456789 -> 0.1234567891234568

### Usage

```tsx
import { NumberInput } from "@reactleaf/input"

function FormComponent() {
  const [value, setValue] = useState(0)

  return <NumberInput label="Number" value={value} onValueChange={(v) => setValue(v)} />
}
```

### Extended Props

> **WARNING** \
> `<NumberInput />` actually use `<input type="text" />` internally, to support number formatting. \
> Reading `event.currentTarget.value` from event listeners would be a text value, not a number. \
> If you want to get numeric value, use `onValueChange` instead.
