# \<NumberInput />

NumberInput is a `react-hook-form` support version of NumberInput.

<Canvas of={Stories.HookForm} />

### Usage

```tsx
import { useForm, FormProvider } from "react-hook-form"
import { NumberInput } from "@reactleaf/input/hookform"

function FormComponent() {
  const form = useForm()

  return (
    <FormProvider {...form}>
      <NumberInput name="amount" label="Amount" placeholder="Placeholder" />
    </FormProvider>
  )
}
```

### Extended Props

This components extended from raw component. Just add some props for `<Controller />` such as `name` or `rules`

> **WARNING** \
> `<NumberInput />` actually use `<input type="text" />` internally, to support number formatting. \
> Reading `event.currentTarget.value` from event listeners would be a text value, not a number. \
> If you want to get numeric value, use `onValueChange` instead.
