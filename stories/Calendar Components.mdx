`@reactleaf/input` wraps the widely-used `react-datepicker` to provide a consistent styling experience. \
By leveraging react-datepicker, we inherit its robust handling of edge cases and comprehensive documentation, ensuring both reliability and ease of use.

So, almost every props from `react-datepicker` is available.

## Prepare Peer Dependencies

```sh
npm install react-datepicker
```
