# \<TextInput />

TextInput is a basic component of `@reactleaf/input`. \
This component simply extends `<input type="text" />`

<Canvas of={TextInputStories.Default} />

### Use in `react-hook-form`

Surely, handle this input as controlled or uncontrolled is up to you. \
So, you can freely choose `register()` or `<Controller />` of `react-hook-form` to handle input in form.

```tsx
import { TextInput } from "@reactleaf/input"

function FormComponent() {
  const { register, formState } = useForm()
  // Easy way using register()
  // return <TextInput label="Name" {...register("name")} errorMessage={formState.error["name"]} />

  return (
    <Controller
      name="name"
      control={control}
      render={({ field, fieldState }) => (
        <TextInput
          label="Name"
          value={field.value}
          onValueChange={field.onChange}
          errorMessage={fieldState.error.message}
        />
      )}
    />
  )
}
```
